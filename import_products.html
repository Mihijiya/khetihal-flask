Import Products from CSV

Instructions:

  • Your CSV file must contain the following columns (case-sensitive): name, description, price, image_url, stock.
  • name, price, and stock are mandatory for each product.
  • price should be a number (e.g., 2.50).
  • stock should be a whole number (e.g., 100).
  • If a product with the same name already exists, its description, price, image_url, and stock will be updated. Otherwise, a new product will be created.
  • image_url should be a path to the image (e.g., `/static_assets/image/product1.jpg`).

Example CSV format:

name,description,price,image_url,stock
Organic Carrots,Fresh organic carrots,1.75,/static_assets/image/carrots.jpg,200
Blueberries,Sweet seedless blueberries,4.00,/static_assets/image/blueberries.jpg,80
